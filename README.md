#  E-Commerce Web Application

##  Project Overview

This project is a full-stack **E-Commerce Web Application** developed using **Java, Spring Boot, Spring Data JPA, MySQL, HTML, CSS, and JavaScript**. It demonstrates the core functionalities of an online shopping platform, including user authentication, product management, shopping cart, checkout, and order management. The application follows a layered architecture (Controller, Service, Repository, Entity) to ensure clean, maintainable, and scalable code.

Users can register, log in, browse products, add items to the shopping cart, review their selected products, proceed to checkout, and place orders. The frontend communicates with the backend through RESTful APIs using the JavaScript Fetch API, while MySQL is used to store user, product, cart, and order information.

##  Features

* User Registration and Login
* Product Listing with Images, Prices, and Stock Details
* Shopping Cart Management
* Add and Remove Products from Cart
* Checkout Process
* Order Placement and Order History
* RESTful API Integration
* Responsive User Interface
* MySQL Database Integration
* Layered Architecture (Controller, Service, Repository, Entity)
